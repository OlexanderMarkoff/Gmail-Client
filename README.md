Написать приложение-клиент к почте Gmail. Приложение выводит почтовые сообщения в виде чатов. На главном экране выводится список собеседников с ограничением по количеству 10. Если есть непрочитанные сообщения, имя собеседника выделяется жирным шрифтом. При касании на имени собеседника открывается окно чата, где слева отображаются сообщения собеседника, а справа - сообщения пользователя. Так как в почтовом чате могут учавствовать несколько собеседников, нужно выводить имя того, кто отправил сообщение. Отображать необходимо только первые 20 символов сообщения, показывать 10 последних сообщений

Требования:
Использовать Gmail API

Помечать сообщения как прочитаные если пользователь открыл чат

Использовать Push уведомления при поступлении новых сообщений

Хранить историю последних 10 сообщений каждого чата для доступа в оффлайн режиме

Опционально:

Использовать рекомендации Material Design

На экране списка чатов загружать дополнительно по 5 чатов при достижении нижнего края списка

Отображение полного текста сообщения по каcанию на короткой версии собщения

Отображение базовых emoji

Не требуется:
Отправка сообщений из клиента
